**RENDER : **

modern cloud Paas, Platformm as a service, designed to make de[loying and hosting web apps, dbs and apis very simple.

instead of configuring complex virtual servers or managing networks urself like aws or gcp, render abstracts all of that away, functions as the modern spiritual successor to older platforms like heroku

CORE CONCEPT : GIT PUSH TO DEPLOY


**What makes render special ?**

- no devops needed : no need to look at terminal command lines, configure security patches or bring ssl certificates for app, render handles auto scaling, routing and system health checks out of the box

- native docker support : if app is too complex for default language selection, simply include dockerfile, render read dockerfile and spin up app fast
